Help


Psinergy.Link functions as a hyperlink (URL) shortener.

Simply paste the link into the URL box, then click 'Shorten'.

After creating a shortened link, a detail page is presented so that you can copy both the Title and Shortened URL together. When you paste it, people will be able to see what that short URL is referring to. It is also for your own reference.

The shortened version of the URL will be presented at the top row on the Recent page. Copy that link then paste it anywhere.

Click 'Recent' to view the most recently shortened URLs.

Click 'Top' to view the most clicked URLs in descending order.

NOTE: Unfortunately, for websites that use Cloudflare or skill testing Captchas, this site is not able to acquire webpage titles at this time. Websites such as ResearchGate, MDPI, IEEE, Wiley, to name a few use these security methods. Because of this, the Title may say "Title not available" for the shortened link. Each link may be edited later, simply grab the title from the original website using your web browser.
