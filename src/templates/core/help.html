Psinergy.Link functions as a hyperlink (URL) shortener.

Click the 'Shorten Link' button, then simply paste the link into the Long URL text box. Then click 'Shorten'.

After creating the shortened link, a detail page is presented so that both the Title and Shortened URL can be copied together. Next, when it is pasted they'll be able to see what the short URL is referring to. It's also for your own reference.

If the URL you are shrinking points directly to a file, such as a .PDF or .JPG, please rename the Title field after the shortened link is created. It will be set to Private until the Title is renamed, and Private field turned off. This is to ensure the link has a proper Title for the content before going public.

All new shortened links are presented at the top row on the All Links page.

Click 'My Links' to view the your most recently shortened URLs.

Click 'All Links' to view everyone's most recently shortened URLs.

The website will inform you if the URL you're trying to shorten was shortened already, either by you or someone else. This is so that the same short link is shared by everyone for the same online content.

When entering tags/topics, enter the text you need for the topic, then press the enter or tab key. A grey badge will surround the topic name. You don't need to enter quotes when entering multiple words for a topic, example: "digital twin".

NOTE: Unfortunately, for websites that use Cloudflare or skill testing Captchas, this site is not able to acquire webpage titles at this time. Websites such as ResearchGate, ScienceDirect, Wiley, YouTube use these security methods or worse. Because of this, the Title may say "Title not available" or "Just a moment...".

Titles can be edited - simply grab the related title from the original website. The public area of the site is cached for 10 minutes. So when you shorten a link, you have about 10 minutes to rename the Title.

Website Features

  • Condenses valid URLs into shareable URLs.
  • The Title of the original website is stored with the shortened URL.
  • The Shortened URL with the Title can be easily copied by clicking a button.
  • Topics (tags) can be added for each shortened link.
  • Titles and Topics can be updated.
  • Shortened URLs can be deleted, if necessary. However, anywhere the link was previously shared will return a "404 Not Found" if clicked.
  • The shortener checks if the original URL has been shortened already. If so, the existing record is shown to you.
  • Shortened URLs are listed by most recent, or by views (most clicked) in descending order.
  • Shortened links are searchable by Title and Topic.
  • For Google/Brave/DuckDuckGo Search, and Google Patent Search, the search query is captured for the Title of the shortened link.
  • Clicking a tag lists all shortened links with that tag.
  • Shortened links can be set to Private. Only you can see your own private links. However, an administrator with database access can see those shortened links. So please stay on point.
  • If a private shortened URL is shared, that short URL will still redirect normally.
  • The blue Alias button can be used to jump to the shortened link and redirected to the original URL.
  • The green Copy button is used to copy the Title, Short URL, and the list of Topics of a shortened link; which can be pasted anywhere for sharing.
  • An RSS feed is provided which contains 50 most recent links.
