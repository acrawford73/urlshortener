FAQ


What does this website do?

The main function is a Hyperlink/URL shortener.

What is a URL shortener?

A URL shortener condenses any valid web address into a shortened one. When the shortened URL is clicked, the browser redirected to the same original web address URL.

To use the shortener, you copy and paste the full original link into this website, then it is condensed into a shareable link.

You may have seen shortened YouTube URLs like: youtu.be/abcd1234. Same idea.

Your shortened links look like this: psinergy.link/abc123. You copy this shortened address then paste it anywhere appropriate.

Why even bother do this?

Psinergist researchers are champions of content sharing; often Google Search URLs. Most of the time those search URLs are massive, ~1000 characters. They take up too much space when posted. So a shortened link is a more "professional" method.

How long do the links persist? Do they expire?

The shortened links do not expire. They'll persist as long as the service is available.

Who is behind this? Can I contact them?

Someone within the Psinergist community. Yes, on Telegram.


Website Features

  • Condenses valid URLs into shortened URLs.
  • The Title of the original website is stored with the shortened URL.
  • The Shortened URL with the Title can be easily copied by clicking a button.
  • Shortened URLs and their titles can be edited.
  • Shortened URLs can be deleted, if necessary.
  • The shortener checks if a URL has been shortened already. If so, the shortened link is shown.
  • Shortened URLs are listed by Recent, or by Top (most clicked) in descending order.
  • Shortened links are searchable by Title.
  • For Google Search and Google Patent Search URLs, the search query is used for the Title of the shortened link.
