FAQ


What does this website do?

The main function is a Hyperlink/URL shortener.

What is a URL shortener?

A URL shortener condenses any valid web address into a shortened one. When the shortened URL is clicked, the browser redirected to the same original web address URL.

To use the shortener, you copy and paste the full original link into this website, then it is condensed into a shareable link.

Shortened links look like this: psinergy.link/abc123. You copy this shortened address then paste it anywhere you wish.

Why even bother do this?

Psinergist researchers are champions of content sharing, often Google Search URLs. Most of the time those search URLs are massive, ~1000 characters, taking up too much space when posted. So a shortened link is a more "professional" method. It also has became a useful ledger of bookmarks.
